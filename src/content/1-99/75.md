# 2 解

```java
class Solution {
    public void sortColors(int[] nums) {
        int num0 = 0;
        int num1 = 0;
        int num2 = 0;
        for (int i = 0; i < nums.length; i++) {
            if (nums[i] == 0) {
                num0++;
            } else if (nums[i] == 1) {
                num1++;
            } else {
                num2++;
            }
        }
        for (int i = 0; i < nums.length; i++) {
            if (num0 != 0) {
                nums[i] = 0;
                num0--;
            } else if (num1 != 0) {
                nums[i] = 1;
                num1--;
            } else if (num2 != 0) {
                nums[i] = 2;
                num2--;
            } else {
                break;
            }
        }
    }
}
```
